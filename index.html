    📱 OPPO Reno11 后台推送完整教程
    ① 开启浏览器通知权限
    在本页点击「第一步：开启通知权限」按钮，系统弹窗选择 允许。
    如已拒绝，需去手机设置里重新开启（见第②步）。
    ② 手机系统通知设置
    手机 设置 → 通知与状态栏 → 找到你正在使用的浏览器（如 OPPO 浏览器 / Chrome / Edge）→ 开启所有通知开关，特别是横幅通知和锁屏通知。
    ③ 关闭电池限制（最关键）
    手机 设置 → 电池 → 耗电保护 / 省电管理 → 找到当前浏览器 → 点击 → 选择 无限制（不要选「智能限制」或「省电」）。
    ⚠️ 这是 ColorOS 后台推送失效的最常见原因。
    ④ 浏览器内部通知授权
    点击地址栏左边的 🔒 锁头 图标 → 通知 → 选择 允许。
    ⑤ 添加到手机桌面（强烈推荐）
    浏览器右上角 菜单（⋮） → 添加到桌面。
    以 PWA 方式从桌面图标打开，系统会赋予它更高的后台存活优先级，通知效果大幅提升。
    ⑥ 验证是否配置成功
    关闭本教程 → 点击「测试后台推送」按钮 → 立刻按 Home 键切出 → 等待约 5 秒 → 如果看到横幅弹出，配置成功！
    ⚠️ 纯前端网页无法做到浏览器完全关闭后推送，请保持浏览器在后台运行（不要手动划掉）。从桌面图标打开 + 关闭电池限制 = 最佳效果。
